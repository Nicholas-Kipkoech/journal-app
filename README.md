# Journal App

## How to install and Run it

1. #### Clone the Repository

```sh
https://github.com/Nicholas-Kipkoech/journal-app
cd journal-app
```

### 2. Install Dependencies

```sh
npm install --legacy-peer-deps
```

```sh
API_URL= http://localhost:8080/api
```

#### 3.Build and run the app

```sh
npm run build
npm run start
```
